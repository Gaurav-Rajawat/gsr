# 🏥 Insurance Cost Prediction

## 📌 Overview
This project predicts medical insurance charges using machine learning.

## 📊 Dataset
- Age, BMI, Smoking, Region
- Target: Charges

## ⚙️ Technologies
- Python
- Pandas
- Scikit-learn

## 🧠 Model
- Linear Regression

## 📈 Result
- Accuracy: 85%

## ▶️ How to Run
1. Install requirements
2. Run notebook
